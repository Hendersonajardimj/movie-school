# Curriculum Suggestions Workspace

Use this file as the team's shared editorial scratchpad.

Purpose:

1. Capture ideas before they become formal lesson edits.
2. Keep brainstorming visible in one place.
3. Turn good ideas into pull requests against `content/weeks/*.md`.

---

## How to use this doc

1. Add suggestions under the correct section.
2. Include your initials and date.
3. Keep entries short and actionable.
4. When an idea is accepted and implemented, mark it as done.

Status tags:

- `[IDEA]` early concept
- `[REVIEW]` needs team feedback
- `[READY]` ready for PR
- `[DONE]` implemented

Entry format:

```md
- [IDEA] (AC, 2026-02-08) Week 03: add one prompt about media framing and fear politics.
  Why: strengthens contextual-awareness outcome.
```
